- **Question 2** (20 marks): 40 minutes - Calculations and strategic analysis

#### Answer Techniques:

**For "Differentiate" Questions:**
1. Define both terms clearly
2. Explain the key difference
3. Provide a real-world example showing the difference

**For Calculation Questions:**
1. Write the formula first
2. Show all steps clearly
3. Box or underline your final answer
4. Include units (e.g., $, %, etc.)

**For "Explain" Questions:**
1. Start with a clear definition
2. Provide 2-3 key points
3. Use real-world examples
4. Conclude with why it matters

**For "List" Questions:**
1. Use bullet points or numbering
2. Be concise but complete
3. Don't elaborate unless asked
